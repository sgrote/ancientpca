# ancientpca
A Probabilistic Approach to Conducting PCA with Sparse Genotype Data from Ancient Samples

## Install and load in R

```
library(devtools)
install_github('sojwolf/ancientpca')
library(ancientpca)
```

## Dependencies

R packages you need installed:

- vcfR
- softImpute
- ggplot2
- ggrepel
- cowplot
